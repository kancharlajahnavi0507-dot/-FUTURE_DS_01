📊 Superstore Sales Dashboard (Power BI)
📌 Project Overview

This project presents an interactive Superstore Sales Dashboard developed using Power BI to analyze retail sales data and uncover meaningful business insights. The dashboard helps stakeholders understand sales performance, profitability, customer behavior, and regional trends through clear and dynamic visualizations.

🎯 Objectives

Analyze overall Sales, Profit, and Quantity

Identify top and bottom performing products

Understand performance by Category, Sub-Category, and Segment

Track monthly and yearly sales trends

Perform region-wise and state-wise analysis

Enable data-driven decision making

📈 Dashboard Highlights

KPI Cards: Total Sales, Total Profit, Total Quantity

Category & Sub-Category wise sales and profit

Segment and customer-level analysis

Regional & state-wise performance map

Time-based trend analysis (Month / Year)

Interactive slicers for better exploration

🛠 Tools & Technologies

Power BI

Power Query (Data Cleaning & Transformation)

DAX (Calculated Measures & KPIs)

CSV Dataset (Superstore Sales Data)

📂 Dataset

Source: Superstore Sales Dataset (CSV format)

Data includes:

Order ID, Order Date, Ship Date

Customer ID, Segment

Category, Sub-Category, Product Name

Sales, Profit, Quantity

Region, State, City

🔧 Data Preparation

Removed duplicates and handled missing values

Converted data types (dates, numbers, text)

Created calculated columns and measures using DAX

Built relationships and optimized the data model
